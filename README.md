Здесь я фиксирую базовые знания о Git и GitHub, которые изучаю на практике.



\## Что такое Git



Git — это распределённая система контроля версий.  

Она позволяет:

\- сохранять историю изменений файлов;

\- возвращаться к предыдущим версиям;

\- работать с проектом локально и синхронизироваться с удалённым репозиторием.

```text

Изменил файлы → git add → git commit → git push



\## Основные команды



\### Инициализация репозитория



git init



\### Проверка состояния



git status



\### Добавление файлов в staging



git add файл

git add .



\### Создание коммита



git commit -m "Сообщение коммита"



\### Просмотр истории



git log



\### Добавление удалённого репозитория



git remote add origin git@github.com:username/repository.git



\### Отправка изменений на GitHub



git push -u origin main



\### Клонирование репозитория



git clone git@github.com:username/repository.git
